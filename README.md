# Demoniac
Demoniac - простая ролевая гира написанная на Python с помощью библиотеки Arcade
## Инструкция по установке
1. Сначала нужно установить необходимые библиотеки из файла requirements.txt
2. Затем откройте терминал из папки с кодом
3. В коммандную строку введите:
```pyinstaller Demoniac.spec```
4. После этого в в папке проекта появятся две папки: dist и build. Папку build можно удалить, а в папке dist юудет лежать готовое приложение.
